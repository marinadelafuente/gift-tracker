# 🎁 Gift Tracker

A cross-platform mobile and web app built with Expo and React Native for tracking gift ideas for the people in your life.

## Features

- Add and manage giftees (recipients) with custom avatars and birthdays
- Track gift ideas per giftee with name, price, description, store, URL, and photo
- Mark gifts as bought, with a "Past Gifts" archive section
- Works on iOS, Android, and (partly) web

## Tech Stack

- [Expo](https://expo.dev/) (managed workflow) with [Expo Router](https://expo.github.io/router/) for file-based navigation
- [React Native](https://reactnative.dev/) + [React Native Web](https://necolas.github.io/react-native-web/)
- [NativeWind](https://www.nativewind.dev/) (Tailwind CSS for React Native)
- [AsyncStorage](https://react-native-async-storage.github.io/async-storage/) for local persistence
- [TanStack Query](https://tanstack.com/query) for data fetching
- [expo-image-picker](https://docs.expo.dev/versions/latest/sdk/imagepicker/) for gift photos

## Getting Started

### Prerequisites

- Node.js 18+
- [Expo CLI](https://docs.expo.dev/get-started/installation/)
- For iOS: Xcode (Mac only)
- For Android: Android Studio

### Install dependencies

```bash
npm install
```

### Run the app

```bash
# Start development server
npm start

# iOS simulator
npm run ios

# Android emulator
npm run android

# Web browser
npm run web
```

### Run with dev client (recommended)

If you have a development build installed on your device:

```bash
npm run start:dev-client
```

## Development Builds

This project uses Expo managed workflow with a dev client. To build the native projects:

```bash
# Generate native folders
npx expo prebuild

# Clean rebuild (use after SDK upgrades or adding native packages)
npx expo prebuild --clean

# Run on device
npx expo run:ios
npx expo run:android
```

> The `/ios` and `/android` folders are git-ignored as they are generated by `prebuild`.

## Project Structure

```
app/
  _layout.tsx          # Root navigation layout
  index.tsx            # Home screen (recipient list)
  recipients/
    [id].tsx           # Recipient detail screen (gift list)
components/
  RecipientList.tsx    # List of giftees
  RecipientModal.tsx   # Add/edit giftee modal
  GiftList.tsx         # List of gift ideas for a giftee
  GiftModal.tsx        # Add/edit gift modal
  GiftItem.tsx         # Individual gift card
  Accordion.tsx        # Collapsible past gifts section
helpers/
  formatDate.ts        # Date formatting utility
  getAvatarUrl.ts      # Avatar URL generator
assets/
  icons/               # SVG icon components
  images/              # App icons and splash screen
```

## Environment

No environment variables are required — the app uses local AsyncStorage for all data persistence.

## Notes

- Camera functionality is not available on web; only image picker is supported
- The date picker uses a native component on iOS/Android
